# StudyBot — AI Course Tutor

AI-powered question-answering system for your course PDFs.
Built with RAG, LangChain, ChromaDB, and Groq Llama3.

## Features
- Upload any course PDF
- Ask questions in natural language
- Get answers with source citations
- Conversation memory across questions
- Analytics dashboard

## Tech Stack
Python · LangChain · ChromaDB · Groq Llama3 · Streamlit · HuggingFace MiniLM
